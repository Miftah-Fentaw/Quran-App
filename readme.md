# Quran App

A beautiful, distraction-free Quran reading application.

## Features

- Clean and elegant reading interface
- Complete Quran text with 114 Surahs
- Offline functionality - no internet required
- Bookmark system for saving your place
- Light and dark themes
- Screenshot capability
- No ads or tracking
- Privacy-focused design

## Installation

Download from your app store and install.

## Usage

Open the app and start reading the Holy Quran with ease.
